Random Word Generator for Anki An Anki add-on for generating random lists of words from input text files. Useful for practicing vocabulary by trying to write a creative story that includes the generated words, or just for finding interesting words you never knew existed! To use with your word list, copy your file containing blank-delimited words to the user_files/wordlists subfolder in the add-on's folder. The file should have the txt extension. Word lists for some languages are included with the add-on in the default_wordlists folder. The add-on can either be used to generate a list of words from the the add-on icon in the note editor and the menu item in the Tools menu of the main screen, or it can be used as a filter in your card templates to dynamically generate different lists each time you see a card. The add-on's custom filter is named randlist and it has some options to customize its behavior: {{randlist words=en start=1000 end=2000 length=10:}}
  • The words option denotes the input file in the wordlists folders you want to generate the list from. It should be written without .txt. The default value is en.
  • start denotes the position in the input list to start randomly choosing words from. The default is 0 (0-based indexing is used).
  • end denotes the position in input list to limit chosen words to the ones before it (the word at index end is excluded). The default is the length of the input list, so words are chosen from the whole list.
  • length denotes the number of words in the generated list. The default is the magic number 7.
The start, end, and length options are useful if you have a word list sorted by frequency (i.e. a frequency list), and only want to get words from a certain range, like the first 5000 words so that you concentrate on learning the most common words first.
